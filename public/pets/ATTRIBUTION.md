# Opencode Pet black cat assets

The shipped `blackcat-*` GIFs are original procedural pixel art generated by `scripts/generate_pet_assets.py` for this project.

Research note: VS Code Pets was evaluated as a reference because it is MIT licensed, but its `media/README.md` states that the cat author asked that cat assets not be freely distributed in GitHub. Those encrypted cat assets are therefore not bundled here.

Design direction: cute black pixel cat, transparent background, OpenCode state overlays, no sound effects.
